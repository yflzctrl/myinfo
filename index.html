сайты

  1. the phrase finder - сайт с 2500 английских идиом, фраз и пословиц, которые мы используем каждый день, с объяснениями их значений и происхождения.
  2. english online - сайт с текстами на английском. в конце каждого текста есть список слов для изучения.
  3. ted talks daily - проект, основанный на знаменитых конференциях и обучающих видео от TED Talks.
  4. snappy words - интерактивный визуальный словарь формирует для каждого слова сеть связанной с ним лексики и фраз.
  5. english central - сайт, где собраны сотни видео разнообразной тематики с субтитрами. после каждого видео есть упражнения для улучшения произношения
  6. exam english - сайт для тех, кто готовится к международным экзаменам по английскому
